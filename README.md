# Student Management System (Console App)

A C# console-based Student Management System that allows users to:

## Features
- Register students
- View student list
- View student details by ID
- Create marksheets
- View student results
- Grade & pass/fail calculation

## Technologies Used
- C#
- .NET Console Application
- LINQ
- Collections (List)

## How to Run
1. Clone the repository
2. Open the project in Visual Studio
3. Run the application using `Ctrl + F5`
